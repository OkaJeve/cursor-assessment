# Cursor IDE and Its Extension Tools

## Tools Installed
* **Cursor IDE**
* **Claude Code for VS Code** by Anthropic from Cursor IDE's extension marketplace
* **Codex - OpenAI's coding agent** from Cursor IDE's extension marketplace
* **Git for Windows**

## Steps Completed
1. Downloaded and installed Cursor IDE from the official website.
2. Installed Claude Code for VS Code extension and Codex - OpenAI's coding agent extension.
3. Created a public GitHub repository.
4. Downloaded and installed Git for Windows from the official website.
5. Opened the repository inside Cursor and set up the terminal environment.
6. Created this documentation, committed the changes, and pushed them to GitHub.

## Issues Encountered and Solutions
* **Issue:** Windows Defender flagged the Cursor installer as unsafe/unrecognized.
  * *Solution:* Verified the digital signature belonged to "Anysphere, Inc." and clicked "Run Anyway".
* **Issue:** Never use GitHub.
  * *Solution:* Searched tutorials online and checked them with Gemini.
